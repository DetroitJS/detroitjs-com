# DetroitJS Website

Powered by [Nextjs](https://github.com/zeit/next.js) and [Tachyons](http://tachyons.io)

# Setup
## Development
`npm i` or `yarn install`

## Production
`npm i --production` or `yarn install --production`

# Usage
## Development
`npm run dev`

## Production
`npm run build` then `npm start`
